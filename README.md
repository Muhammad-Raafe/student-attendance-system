# 📚 Student Attendance System

A beginner-friendly **Student Attendance System** built in Python.
This project helps practice core programming concepts like **functions, loops, and list of dictionaries**.

---

## 🚀 Features

* ➕ Add Student
* 📋 View All Students
* 🔍 Search Student
* ✅ Count Present Students

---

## 🧠 Concepts Used

* Functions
* Lists & Dictionaries
* List of Dictionaries
* Loops (for loop)
* Conditional Statements (if-else)
* User Input Handling

---

## 🛠️ How It Works

The system stores student data using a **list of dictionaries**, where each student has:

* Name
* Attendance Status (Present/Absent)

Users can interact with the program using a simple menu system.

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the file:

```bash
python student_attendance.py
```

## 📌 Example

```
1/ add student
2/ view students
3/ search student
4/ total present
5/ exit
```
